# BHUC Clinician Directory

> Sample clinician profiles (specialty, credentials, availability) for the BHUC Scheduling Agent's clinician-matching retrieval. Demo data consistent with the BHUC app.


## Directory overview and matching criteria

<p><em>Sample/demo data authored for the BHUC reference build — not a real payer or provider record. Kept consistent with the BHUC app fixtures.</em></p><p><strong>How clinician matching works</strong></p><p>The BHUC Scheduling Agent recommends a clinician using only clinical need, clinician credentials/specialty, and availability. It is blind to race, ethnicity, gender identity, ZIP code, insurance type, and any other protected or proxy demographic field — these are excluded from the matching inputs, and a fairness check must pass before any recommendation is surfaced. Proposals are drafts for staff to confirm; the agent never books autonomously.</p>
